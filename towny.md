## /towny

| Команда | Описание |
| :--- | :--- |
| `/towny` | Показывает список базовых команд Towny. |
| `/towny ?` | Показывает расширенный список команд Towny. |
| `/towny allowedblocks` | Открывает меню с блоками, которые разрешено ставить/ломать. |
| `/towny itemuse` | Показывает предметы из списка `item_use_ids`. |
| `/towny map` | Показывает карту Towny. |
| `/towny hud` | Включает отображение карты в интерфейсе игрока (scoreboard). |
| `/towny plotclearblocks` | Показывает блоки, удаляемые командой `/plot clear`. |
| `/towny prices` | Показывает налоги и стоимость содержания города. |
| `/towny switches` | Показывает блоки из списка `switch_ids`. |
| `/towny time` | Показывает время до следующего сбора налогов/обслуживания. |
| `/towny top residents {all/town/nation}` | Показывает топ жителей (по всем, по городу, по нации). |
| `/towny top land {all/resident/town}` | Показывает топ землевладельцев (все, резиденты, города). |
| `/towny spy` | Команда администратора для слежки за всеми чатами. |
| `/towny tree` | Показывает разнообразную техническую информацию. |
| `/towny universe` | Показывает полную статистику Towny (жители, города, нации, миры, занятые участки). |
| `/towny v` | Показывает версию Towny. |
| `/towny wildsblocks` | Показывает блоки, которые можно использовать на диких участках и в дикой природе. |

## /plot

### Основные команды

| Команда | Описание |
| :--- | :--- |
| `/plot` | Показывает список команд `/plot`. |
| `/plot claim` | Приватизировать участок, выставленный на продажу (команда резидента). |
| `/plot auto` | Автоматически приватизировать несколько участков вокруг (если они продаются). |
| `/plot unclaim` | Отказаться от личного участка (резидент). |
| `/plot unclaim circle {радиус}` | Отказаться от личных участков в форме круга (радиус от текущей позиции). |
| `/plot unclaim rect {радиус}` | Отказаться от личных участков в форме прямоугольника. |
| `/plot forsale` / `fs {цена}` | Выставить участок на продажу за указанную цену. |
| `/plot forsale circle {радиус} {цена}` | Выставить на продажу участки в форме круга. |
| `/plot forsale rect {радиус} {цена}` | Выставить на продажу участки в форме прямоугольника. |
| `/plot notforsale` / `nfs` | Снять участок с продажи. |
| `/plot nfs circle {радиус}` | Снять с продажи участки в форме круга. |
| `/plot nfs rect {радиус}` | Снять с продажи участки в форме прямоугольника. |
| `/plot evict` | Выселить владельца с участка (обычно используется мэром). |
| `/plot evict forsale` | Выселить владельца и сразу выставить участок на продажу. |

### Работа с районами (`/plot district`)

| Команда | Описание |
| :--- | :--- |
| `/plot district create` / `new [имя]` | Создать новый район с указанным именем. |
| `/plot district add [имя]` | Добавить текущий участок в район. Если активно добавление, имя можно не повторять. |
| `/plot district rename [имя]` | Переименовать район. |
| `/plot district remove` | Убрать текущий участок из района. |
| `/plot district delete` | Удалить район, в котором вы стоите. |

### Доверие и права на участке (`/plot trust`, `/plot perm`)

| Команда | Описание |
| :--- | :--- |
| `/plot trust add {игрок}` | Добавить игрока в доверенные на этом участке. |
| `/plot trust remove {игрок}` | Убрать игрока из доверенных на этом участке. |
| `/plot perm` | Показать строку прав (`perm`) участка, на котором стоите. |
| `/plot perm gui` | Открыть графический интерфейс настройки персональных прав участка. |
| `/plot perm add {игрок}` | Добавить игрока в список персонального переопределения прав на участке. |
| `/plot perm remove {игрок}` | Убрать игрока из списка переопределения прав. |
| `/plot perm hud` | Включить/выключить отображение строки прав на экране (scoreboard). |

### Установка типа участка (`/plot set`)

| Команда | Описание |
| :--- | :--- |
| `/plot set reset` | Вернуть тип участка к обычному (сбрасывает shop/embassy/arena/wilds). |
| `/plot set shop` | Установить участок как магазин. |
| `/plot set embassy` | Установить участок как посольство. |
| `/plot set arena` | Установить участок как арену. |
| `/plot set wilds` | Установить участок как дикий (wilds). |
| `/plot set inn` | Установить участок как гостиницу. |
| `/plot set jail` | Установить участок как тюрьму. |
| `/plot set farm` | Установить участок как ферму. |
| `/plot set bank` | Установить участок как банк. |
| `/plot set outpost` | Установить участок как форпост (стоимость как у `/t claim outpost`). |
| `/plot set spawn` | Установить точку спавна внутри существующего форпоста на ваше текущее место. |
| `/plot set name` | Переименовать участок (мэр или владелец). Заменяет стандартную надпись «~Unowned». |

### Права на участке (`/plot set perm`)

| Команда | Описание |
| :--- | :--- |
| `/plot set perm on/off` | Включить/выключить переопределение прав на конкретном участке. |
| `/plot set perm {resident/ally/outsider} on/off` | Включить/выключить права для всех действий у указанной группы. |
| `/plot set perm {build/destroy/switch/itemuse} on/off` | Включить/выключить конкретное действие для всех групп. |
| `/plot set perm {resident/ally/outsider} {build/destroy/switch/itemuse} on/off` | Точечная настройка конкретного действия для конкретной группы. |
| `/plot set perm reset` | Сбросить права участка к стандартным (из `/town` или `/resident`). |

### Ограничения по дням (`/plot set minjoindays`, `/plot set maxjoindays`)

| Команда | Описание |
| :--- | :--- |
| `/plot set minjoindays {число}` | Установить минимальное количество дней в городе для привата участка. |
| `/plot set minjoindays clear` | Убрать ограничение минимальных дней. |
| `/plot set maxjoindays {число}` | Установить максимальное количество дней в городе для привата участка. |
| `/plot set maxjoindays clear` | Убрать ограничение максимальных дней. |

### Переключатели участка (`/plot toggle`)

| Команда | Описание |
| :--- | :--- |
| `/plot toggle fire` | Вкл/выкл распространение огня на участке. |
| `/plot toggle pvp` | Вкл/выкл PvP на участке. |
| `/plot toggle explosion` | Вкл/выкл взрывы на участке. |
| `/plot toggle mob` | Вкл/выкл появление враждебных мобов на участке. |
| `/plot toggle taxed` | Вкл/выкл налог для личного владельца участка. |

### Очистка участка (`/plot clear`)

| Команда | Описание |
| :--- | :--- |
| `/plot clear` | Удалить блоки из списка `plotclearblocks` на участке (мэр на городской земле, или владелец на личном). |

### Работа с группами участков (`/plot group`)

| Команда | Описание |
| :--- | :--- |
| `/plot group add/new/create {имя}` | Создать группу участков или добавить текущий участок в существующую группу. |
| `/plot group remove` | Убрать текущий участок из группы. |
| `/plot group delete` | Полностью удалить группу участков. |
| `/plot group rename {новое_имя}` | Переименовать группу участков. |
| `/plot group set {тип}` | Установить тип для всей группы (кроме Jail). |
| `/plot group set perm ...` | Настроить права всей группы (синтаксис как у `/plot set perm`). |
| `/plot group toggle ...` | Применить переключатели ко всей группе (как `/plot toggle`). |
| `/plot group forsale/fs {цена}` | Выставить всю группу на продажу. |
| `/plot group notforsale/nfs` | Снять группу с продажи. |

#### Доверие и права внутри группы (`/plot group trust`, `/plot group perm`)

| Команда | Описание |
| :--- | :--- |
| `/plot group trust add {игрок}` | Добавить игрока в доверенные для всей группы. |
| `/plot group trust remove {игрок}` | Убрать игрока из доверенных группы. |
| `/plot group perm gui` | Открыть GUI для настройки персональных прав группы. |
| `/plot group perm add {игрок}` | Добавить игрока в переопределение прав группы. |
| `/plot group perm remove {игрок}` | Убрать игрока из переопределения прав группы. |

### Тюремные камеры (`/plot jailcell`)

| Команда | Описание |
| :--- | :--- |
| `/plot jailcell add` | Добавить тюремную камеру на тюремном участке в месте, где вы стоите. |
| `/plot jailcell remove` | Удалить тюремную камеру с тюремного участка. |

## /resident

### Основные команды

| Команда | Описание |
| :--- | :--- |
| `/resident` | Показывает экран резидента. |
| `/resident ?` | Показывает доступные команды `/res`. |
| `/resident {резидент}` | Показывает экран другого резидента. |

### Друзья (`/resident friend`)

| Команда | Описание |
| :--- | :--- |
| `/resident friend add {игрок} .. {игрок}` | Добавить онлайн-игрока(ов) в список друзей. |
| `/resident friend add+ {игрок} .. {игрок}` | Добавить офлайн-игрока(ов) в список друзей. |
| `/resident friend remove {игрок} .. {игрок}` | Удалить онлайн-игрока(ов) из друзей. |
| `/resident friend remove+ {игрок} .. {игрок}` | Удалить офлайн-игрока(ов) из друзей. |
| `/resident friend clearlist` | Удалить всех друзей из списка. |
| `/resident friend list` | Показать список ваших друзей. |
| `/resident friend list online` | Показать список друзей, которые сейчас онлайн. |

### Список резидентов, тюрьма и вне закона

| Команда | Описание |
| :--- | :--- |
| `/resident list` | Показать список резидентов из папки данных Towny, которые онлайн. |
| `/resident jail paybail` | Заплатить штраф за выход из тюрьмы. Деньги получает город, владеющий тюрьмой. |
| `/resident outlawlist [игрок]` | Показать список городов, где указанный игрок объявлен вне закона. Без имени — свои города. |

### Участки резидента (`/resident plotlist`)

| Команда | Описание |
| :--- | :--- |
| `/resident plotlist [страница]` | Показать список своих личных участков. Можно указать номер страницы. |
| `/resident plotlist [имя] [страница]` | Показать список участков указанного игрока. |

### Спавн (`/resident spawn`)

| Команда | Описание |
| :--- | :--- |
| `/resident spawn` | Телепортирует к своей кровати (если `deny_bed_use: true` и у игрока установлен спавн в кровати). |

### Переключатели (`/resident toggle`)

| Команда | Описание |
| :--- | :--- |
| `/resident toggle pvp` | Вкл/выкл PvP на всех личных участках резидента. |
| `/resident toggle fire` | Вкл/выкл распространение огня на всех личных участках. |
| `/resident toggle explosion` | Вкл/выкл взрывы на всех личных участках. |
| `/resident toggle mobs` | Вкл/выкл враждебных мобов на всех личных участках. |
| `/resident toggle map` | Включает карту Towny, обновляющуюся при пересечении границ участков. |
| `/resident toggle townclaim` | Режим авто-привата: при ходьбе автоматически выполняется `/t claim`. |
| `/resident toggle townunclaim` | Режим авто-отказа: при ходьбе автоматически выполняется `/t unclaim`. |
| `/resident toggle bedspawn` | Режим предпочтения спавна у кровати, а не в городе. |
| `/resident toggle plotborder` | Вкл/выкл дымчатую границу участка при переходе между чанками. |
| `/resident toggle constantplotborder` | Вкл постоянную дымчатую границу участков (не исчезает). |
| `/resident toggle townborder` | Вкл постоянную дымчатую границу того города, в котором стоите. |
| `/resident toggle bordertitles` | Вкл/выкл сообщения-заголовки при входе/выходе из города (если `using_titles: true` в конфиге). |
| `/resident toggle plotgroup` | Включает режим автоматического добавления участков в строящуюся группу участков при заходе на них. |
| `/resident toggle district` | Включает режим автоматического добавления участков в строящийся район при заходе на них. |
| `/resident toggle spy` | Для администраторов: включить слежку за всеми чат-каналами. |
| `/resident toggle ignoreinvites` | Вкл/выкл игнорирование приглашений в города. |
| `/resident toggle ignoreplots` | Вкл/выкл игнорирование уведомлений об участках в городе. |
| `/resident toggle reset` | Выключает все активные режимы. |
| `/resident toggle infotool` | Для администраторов: позволяет использовать глиняный кирпич для получения Bukkit-имён блоков и сущностей. |
| `/resident toggle adminbypass` | Для администраторов: игнорировать ограничения разрешений (режим «как не админ»). |

### Настройка прав (`/resident set perm`)

| Команда | Описание |
| :--- | :--- |
| `/resident set perm on/off` | Включить/выключить переопределение прав на экране резидента. |
| `/resident set perm {friend/ally/outsider} on/off` | Вкл/выкл все права для указанной группы. |
| `/resident set perm {build/destroy/switch/itemuse} on/off` | Вкл/выкл конкретное действие для всех. |
| `/resident set perm {friend/ally/outsider} {build/destroy/switch/itemuse} on/off` | Точечная настройка конкретного действия для конкретной группы. |
| `/resident set perm reset` | Применить стандартную строку прав (из `/resident`) ко всем своим личным участкам. |

### Статус «О себе» (`/resident about`)

| Команда | Описание |
| :--- | :--- |
| `/resident about {сообщение}` | Установить текст «о себе», отображаемый в `/res`. |
| `/resident about clear` | Удаляет все ваши активные режимы. |
| `/resident about reset` | Удаляет все ваши режимы и заново присваивает стандартные. |

### Налоги (`/resident tax`)

| Команда | Описание |
| :--- | :--- |
| `/resident tax [игрок]` | Показать сумму налогов, уплачиваемых игроком (если игрок не указан — свои налоги). |

## /town

### Основные команды и информация

| Команда | Описание |
| :--- | :--- |
| `/town` | Показывает экран вашего города. |
| `/town ?` | Показывает доступные команды `/town`. |
| `/town {город}` | Показывает экран другого города. |
| `/town here` | Показывает экран города, в котором вы стоите. |
| `/town online` | Показывает жителей вашего города, которые сейчас в сети. |
| `/town plots {город}` | Показывает список участков, их типы и доход, принадлежащих городу. |
| `/town reslist {город}` | Показывает ПОЛНЫЙ список всех жителей города. |
| `/town say {сообщение}` | Отправляет сообщение всем онлайн-членам города. |
| `/town allylist {город}` | Показывает список союзников города. |
| `/town enemylist {город}` | Показывает список врагов города. |
| `/town nearby` | Показывает до 10 ближайших городов. |
| `/town leave` | Покинуть город. |
| `/town reclaim` | Позволяет жителю восстановить свой разрушенный город. |
| `/town merge {город}` | Присоединить указанный город к вашему (указанный город вливается в ваш). |
| `/town delete {город}` | Команда админа/мэра для удаления города из файлов Towny. |

### Создание города и управление жителями

| Команда | Описание |
| :--- | :--- |
| `/town new {название}` | Создать новый город. |
| `/town add {житель} .. {житель}` | Мэр добавляет жителей в город. |
| `/town kick {житель} .. {житель}` | Мэр выгоняет жителей из города. |
| `/town join {город}` | Вступить в открытый город (не требующий приглашений). |
| `/town purge {дни}` | Выгнать жителей, неактивных указанное количество дней (кроме мэров и NPC). |

### Спавн

| Команда | Описание |
| :--- | :--- |
| `/town spawn` | Телепортирует на спавн вашего города. |
| `/town spawn {город}` | Телепортирует на спавн другого города. |

### Список городов (`/town list`)

| Команда | Описание |
| :--- | :--- |
| `/town list by name {стр}` | Сортировка по алфавиту. |
| `/town list by resident {стр}` | Сортировка по количеству жителей (больше — выше). |
| `/town list by balance {стр}` | Сортировка по балансу банка нации. |
| `/town list by townblocks {стр}` | Сортировка по количеству захваченных участков. |
| `/town list by forsale {стр}` | Сортировка по выставленным на продажу городам, по цене. |
| `/town list by online {стр}` | Сортировка по количеству игроков онлайн. |
| `/town list by open {стр}` | Сначала открытые города, затем по убыванию жителей. |
| `/town list by public {стр}` | Сначала публичные города, затем по убыванию жителей. |
| `/town list by ruined {стр}` | Сначала разрушенные города, затем по убыванию жителей. |
| `/town list by bankrupt {стр}` | Сначала обанкротившиеся города, затем по убыванию жителей. |
| `/town list by founded {стр}` | Сортировка по дате основания, старые сначала. |

### Управление участками

| Команда | Описание |
| :--- | :--- |
| `/town claim` | Приватизировать участок, на котором стоите, для города. |
| `/town claim outpost <#\|{имя}\|{имя:#}>` | Приватизировать форпост. `{имя}` — по названию, `{имя:#}` — если имя начинается с цифры. |
| `/town claim {радиус}` | Приватизировать область участков вокруг (радиус от текущей позиции). |
| `/town claim auto` | Автоматически приватизировать максимум участков вокруг, исходя из денег в банке и доступных участков. |
| `/town claim fill` | Заполнить окружённый участками регион внутри границ (после того, как граница захвачена). |
| `/town unclaim` | Отказаться от участка, на котором стоите (команда мэра). |
| `/town unclaim all` | Отказаться от всех городских участков. |
| `/town unclaim {радиус}` | Отказаться от области участков вокруг. |
| `/town unclaim outpost` | Исправить глючные форпосты (для MySQL серверов до версии 0.92.0.0). |
