## /towny

| Команда | Описание |
| :--- | :--- |
| `/towny` | Показывает список базовых команд Towny. |
| `/towny ?` | Показывает расширенный список команд Towny. |
| `/towny allowedblocks` | Открывает меню с блоками, которые разрешено ставить/ломать. |
| `/towny itemuse` | Показывает предметы из списка `item_use_ids`. |
| `/towny map` | Показывает карту Towny. |
| `/towny hud` | Включает отображение карты в интерфейсе игрока (scoreboard). |
| `/towny plotclearblocks` | Показывает блоки, удаляемые командой `/plot clear`. |
| `/towny prices` | Показывает налоги и стоимость содержания города. |
| `/towny switches` | Показывает блоки из списка `switch_ids`. |
| `/towny time` | Показывает время до следующего сбора налогов/обслуживания. |
| `/towny top residents {all/town/nation}` | Показывает топ жителей (по всем, по городу, по нации). |
| `/towny top land {all/resident/town}` | Показывает топ землевладельцев (все, резиденты, города). |
| `/towny spy` | Команда администратора для слежки за всеми чатами. |
| `/towny tree` | Показывает разнообразную техническую информацию. |
| `/towny universe` | Показывает полную статистику Towny (жители, города, нации, миры, занятые участки). |
| `/towny v` | Показывает версию Towny. |
| `/towny wildsblocks` | Показывает блоки, которые можно использовать на диких участках и в дикой природе. |

## /plot

### Основные команды

| Команда | Описание |
| :--- | :--- |
| `/plot` | Показывает список команд `/plot`. |
| `/plot claim` | Приватизировать участок, выставленный на продажу (команда резидента). |
| `/plot auto` | Автоматически приватизировать несколько участков вокруг (если они продаются). |
| `/plot unclaim` | Отказаться от личного участка (резидент). |
| `/plot unclaim circle {радиус}` | Отказаться от личных участков в форме круга (радиус от текущей позиции). |
| `/plot unclaim rect {радиус}` | Отказаться от личных участков в форме прямоугольника. |
| `/plot forsale` / `fs {цена}` | Выставить участок на продажу за указанную цену. |
| `/plot forsale circle {радиус} {цена}` | Выставить на продажу участки в форме круга. |
| `/plot forsale rect {радиус} {цена}` | Выставить на продажу участки в форме прямоугольника. |
| `/plot notforsale` / `nfs` | Снять участок с продажи. |
| `/plot nfs circle {радиус}` | Снять с продажи участки в форме круга. |
| `/plot nfs rect {радиус}` | Снять с продажи участки в форме прямоугольника. |
| `/plot evict` | Выселить владельца с участка (обычно используется мэром). |
| `/plot evict forsale` | Выселить владельца и сразу выставить участок на продажу. |

### Работа с районами (`/plot district`)

| Команда | Описание |
| :--- | :--- |
| `/plot district create` / `new [имя]` | Создать новый район с указанным именем. |
| `/plot district add [имя]` | Добавить текущий участок в район. Если активно добавление, имя можно не повторять. |
| `/plot district rename [имя]` | Переименовать район. |
| `/plot district remove` | Убрать текущий участок из района. |
| `/plot district delete` | Удалить район, в котором вы стоите. |

### Доверие и права на участке (`/plot trust`, `/plot perm`)

| Команда | Описание |
| :--- | :--- |
| `/plot trust add {игрок}` | Добавить игрока в доверенные на этом участке. |
| `/plot trust remove {игрок}` | Убрать игрока из доверенных на этом участке. |
| `/plot perm` | Показать строку прав (`perm`) участка, на котором стоите. |
| `/plot perm gui` | Открыть графический интерфейс настройки персональных прав участка. |
| `/plot perm add {игрок}` | Добавить игрока в список персонального переопределения прав на участке. |
| `/plot perm remove {игрок}` | Убрать игрока из списка переопределения прав. |
| `/plot perm hud` | Включить/выключить отображение строки прав на экране (scoreboard). |

### Установка типа участка (`/plot set`)

| Команда | Описание |
| :--- | :--- |
| `/plot set reset` | Вернуть тип участка к обычному (сбрасывает shop/embassy/arena/wilds). |
| `/plot set shop` | Установить участок как магазин. |
| `/plot set embassy` | Установить участок как посольство. |
| `/plot set arena` | Установить участок как арену. |
| `/plot set wilds` | Установить участок как дикий (wilds). |
| `/plot set inn` | Установить участок как гостиницу. |
| `/plot set jail` | Установить участок как тюрьму. |
| `/plot set farm` | Установить участок как ферму. |
| `/plot set bank` | Установить участок как банк. |
| `/plot set outpost` | Установить участок как форпост (стоимость как у `/t claim outpost`). |
| `/plot set spawn` | Установить точку спавна внутри существующего форпоста на ваше текущее место. |
| `/plot set name` | Переименовать участок (мэр или владелец). Заменяет стандартную надпись «~Unowned». |

### Права на участке (`/plot set perm`)

| Команда | Описание |
| :--- | :--- |
| `/plot set perm on/off` | Включить/выключить переопределение прав на конкретном участке. |
| `/plot set perm {resident/ally/outsider} on/off` | Включить/выключить права для всех действий у указанной группы. |
| `/plot set perm {build/destroy/switch/itemuse} on/off` | Включить/выключить конкретное действие для всех групп. |
| `/plot set perm {resident/ally/outsider} {build/destroy/switch/itemuse} on/off` | Точечная настройка конкретного действия для конкретной группы. |
| `/plot set perm reset` | Сбросить права участка к стандартным (из `/town` или `/resident`). |

### Ограничения по дням (`/plot set minjoindays`, `/plot set maxjoindays`)

| Команда | Описание |
| :--- | :--- |
| `/plot set minjoindays {число}` | Установить минимальное количество дней в городе для привата участка. |
| `/plot set minjoindays clear` | Убрать ограничение минимальных дней. |
| `/plot set maxjoindays {число}` | Установить максимальное количество дней в городе для привата участка. |
| `/plot set maxjoindays clear` | Убрать ограничение максимальных дней. |

### Переключатели участка (`/plot toggle`)

| Команда | Описание |
| :--- | :--- |
| `/plot toggle fire` | Вкл/выкл распространение огня на участке. |
| `/plot toggle pvp` | Вкл/выкл PvP на участке. |
| `/plot toggle explosion` | Вкл/выкл взрывы на участке. |
| `/plot toggle mob` | Вкл/выкл появление враждебных мобов на участке. |
| `/plot toggle taxed` | Вкл/выкл налог для личного владельца участка. |

### Очистка участка (`/plot clear`)

| Команда | Описание |
| :--- | :--- |
| `/plot clear` | Удалить блоки из списка `plotclearblocks` на участке (мэр на городской земле, или владелец на личном). |

### Работа с группами участков (`/plot group`)

| Команда | Описание |
| :--- | :--- |
| `/plot group add/new/create {имя}` | Создать группу участков или добавить текущий участок в существующую группу. |
| `/plot group remove` | Убрать текущий участок из группы. |
| `/plot group delete` | Полностью удалить группу участков. |
| `/plot group rename {новое_имя}` | Переименовать группу участков. |
| `/plot group set {тип}` | Установить тип для всей группы (кроме Jail). |
| `/plot group set perm ...` | Настроить права всей группы (синтаксис как у `/plot set perm`). |
| `/plot group toggle ...` | Применить переключатели ко всей группе (как `/plot toggle`). |
| `/plot group forsale/fs {цена}` | Выставить всю группу на продажу. |
| `/plot group notforsale/nfs` | Снять группу с продажи. |

#### Доверие и права внутри группы (`/plot group trust`, `/plot group perm`)

| Команда | Описание |
| :--- | :--- |
| `/plot group trust add {игрок}` | Добавить игрока в доверенные для всей группы. |
| `/plot group trust remove {игрок}` | Убрать игрока из доверенных группы. |
| `/plot group perm gui` | Открыть GUI для настройки персональных прав группы. |
| `/plot group perm add {игрок}` | Добавить игрока в переопределение прав группы. |
| `/plot group perm remove {игрок}` | Убрать игрока из переопределения прав группы. |

### Тюремные камеры (`/plot jailcell`)

| Команда | Описание |
| :--- | :--- |
| `/plot jailcell add` | Добавить тюремную камеру на тюремном участке в месте, где вы стоите. |
| `/plot jailcell remove` | Удалить тюремную камеру с тюремного участка. |

## /resident

### Основные команды

| Команда | Описание |
| :--- | :--- |
| `/resident` | Показывает экран резидента. |
| `/resident ?` | Показывает доступные команды `/res`. |
| `/resident {резидент}` | Показывает экран другого резидента. |

### Друзья (`/resident friend`)

| Команда | Описание |
| :--- | :--- |
| `/resident friend add {игрок} .. {игрок}` | Добавить онлайн-игрока(ов) в список друзей. |
| `/resident friend add+ {игрок} .. {игрок}` | Добавить офлайн-игрока(ов) в список друзей. |
| `/resident friend remove {игрок} .. {игрок}` | Удалить онлайн-игрока(ов) из друзей. |
| `/resident friend remove+ {игрок} .. {игрок}` | Удалить офлайн-игрока(ов) из друзей. |
| `/resident friend clearlist` | Удалить всех друзей из списка. |
| `/resident friend list` | Показать список ваших друзей. |
| `/resident friend list online` | Показать список друзей, которые сейчас онлайн. |

### Список резидентов, тюрьма и вне закона

| Команда | Описание |
| :--- | :--- |
| `/resident list` | Показать список резидентов из папки данных Towny, которые онлайн. |
| `/resident jail paybail` | Заплатить штраф за выход из тюрьмы. Деньги получает город, владеющий тюрьмой. |
| `/resident outlawlist [игрок]` | Показать список городов, где указанный игрок объявлен вне закона. Без имени — свои города. |

### Участки резидента (`/resident plotlist`)

| Команда | Описание |
| :--- | :--- |
| `/resident plotlist [страница]` | Показать список своих личных участков. Можно указать номер страницы. |
| `/resident plotlist [имя] [страница]` | Показать список участков указанного игрока. |

### Спавн (`/resident spawn`)

| Команда | Описание |
| :--- | :--- |
| `/resident spawn` | Телепортирует к своей кровати (если `deny_bed_use: true` и у игрока установлен спавн в кровати). |

### Переключатели (`/resident toggle`)

| Команда | Описание |
| :--- | :--- |
| `/resident toggle pvp` | Вкл/выкл PvP на всех личных участках резидента. |
| `/resident toggle fire` | Вкл/выкл распространение огня на всех личных участках. |
| `/resident toggle explosion` | Вкл/выкл взрывы на всех личных участках. |
| `/resident toggle mobs` | Вкл/выкл враждебных мобов на всех личных участках. |
| `/resident toggle map` | Включает карту Towny, обновляющуюся при пересечении границ участков. |
| `/resident toggle townclaim` | Режим авто-привата: при ходьбе автоматически выполняется `/t claim`. |
| `/resident toggle townunclaim` | Режим авто-отказа: при ходьбе автоматически выполняется `/t unclaim`. |
| `/resident toggle bedspawn` | Режим предпочтения спавна у кровати, а не в городе. |
| `/resident toggle plotborder` | Вкл/выкл дымчатую границу участка при переходе между чанками. |
| `/resident toggle constantplotborder` | Вкл постоянную дымчатую границу участков (не исчезает). |
| `/resident toggle townborder` | Вкл постоянную дымчатую границу того города, в котором стоите. |
| `/resident toggle bordertitles` | Вкл/выкл сообщения-заголовки при входе/выходе из города (если `using_titles: true` в конфиге). |
| `/resident toggle plotgroup` | Включает режим автоматического добавления участков в строящуюся группу участков при заходе на них. |
| `/resident toggle district` | Включает режим автоматического добавления участков в строящийся район при заходе на них. |
| `/resident toggle spy` | Для администраторов: включить слежку за всеми чат-каналами. |
| `/resident toggle ignoreinvites` | Вкл/выкл игнорирование приглашений в города. |
| `/resident toggle ignoreplots` | Вкл/выкл игнорирование уведомлений об участках в городе. |
| `/resident toggle reset` | Выключает все активные режимы. |
| `/resident toggle infotool` | Для администраторов: позволяет использовать глиняный кирпич для получения Bukkit-имён блоков и сущностей. |
| `/resident toggle adminbypass` | Для администраторов: игнорировать ограничения разрешений (режим «как не админ»). |

### Настройка прав (`/resident set perm`)

| Команда | Описание |
| :--- | :--- |
| `/resident set perm on/off` | Включить/выключить переопределение прав на экране резидента. |
| `/resident set perm {friend/ally/outsider} on/off` | Вкл/выкл все права для указанной группы. |
| `/resident set perm {build/destroy/switch/itemuse} on/off` | Вкл/выкл конкретное действие для всех. |
| `/resident set perm {friend/ally/outsider} {build/destroy/switch/itemuse} on/off` | Точечная настройка конкретного действия для конкретной группы. |
| `/resident set perm reset` | Применить стандартную строку прав (из `/resident`) ко всем своим личным участкам. |

### Статус «О себе» (`/resident about`)

| Команда | Описание |
| :--- | :--- |
| `/resident about {сообщение}` | Установить текст «о себе», отображаемый в `/res`. |
| `/resident about clear` | Удаляет все ваши активные режимы. |
| `/resident about reset` | Удаляет все ваши режимы и заново присваивает стандартные. |

### Налоги (`/resident tax`)

| Команда | Описание |
| :--- | :--- |
| `/resident tax [игрок]` | Показать сумму налогов, уплачиваемых игроком (если игрок не указан — свои налоги). |

## /town

### Основные команды и информация

| Команда | Описание |
| :--- | :--- |
| `/town` | Показывает экран вашего города. |
| `/town ?` | Показывает доступные команды `/town`. |
| `/town {город}` | Показывает экран другого города. |
| `/town here` | Показывает экран города, в котором вы стоите. |
| `/town online` | Показывает жителей вашего города, которые сейчас в сети. |
| `/town plots {город}` | Показывает список участков, их типы и доход, принадлежащих городу. |
| `/town reslist {город}` | Показывает ПОЛНЫЙ список всех жителей города. |
| `/town say {сообщение}` | Отправляет сообщение всем онлайн-членам города. |
| `/town allylist {город}` | Показывает список союзников города. |
| `/town enemylist {город}` | Показывает список врагов города. |
| `/town nearby` | Показывает до 10 ближайших городов. |
| `/town leave` | Покинуть город. |
| `/town reclaim` | Позволяет жителю восстановить свой разрушенный город. |
| `/town merge {город}` | Присоединить указанный город к вашему (указанный город вливается в ваш). |
| `/town delete {город}` | Команда админа/мэра для удаления города из файлов Towny. |

### Создание города и управление жителями

| Команда | Описание |
| :--- | :--- |
| `/town new {название}` | Создать новый город. |
| `/town add {житель} .. {житель}` | Мэр добавляет жителей в город. |
| `/town kick {житель} .. {житель}` | Мэр выгоняет жителей из города. |
| `/town join {город}` | Вступить в открытый город (не требующий приглашений). |
| `/town purge {дни}` | Выгнать жителей, неактивных указанное количество дней (кроме мэров и NPC). |

### Спавн

| Команда | Описание |
| :--- | :--- |
| `/town spawn` | Телепортирует на спавн вашего города. |
| `/town spawn {город}` | Телепортирует на спавн другого города. |

### Список городов (`/town list`)

| Команда | Описание |
| :--- | :--- |
| `/town list by name {стр}` | Сортировка по алфавиту. |
| `/town list by resident {стр}` | Сортировка по количеству жителей (больше — выше). |
| `/town list by balance {стр}` | Сортировка по балансу банка нации. |
| `/town list by townblocks {стр}` | Сортировка по количеству захваченных участков. |
| `/town list by forsale {стр}` | Сортировка по выставленным на продажу городам, по цене. |
| `/town list by online {стр}` | Сортировка по количеству игроков онлайн. |
| `/town list by open {стр}` | Сначала открытые города, затем по убыванию жителей. |
| `/town list by public {стр}` | Сначала публичные города, затем по убыванию жителей. |
| `/town list by ruined {стр}` | Сначала разрушенные города, затем по убыванию жителей. |
| `/town list by bankrupt {стр}` | Сначала обанкротившиеся города, затем по убыванию жителей. |
| `/town list by founded {стр}` | Сортировка по дате основания, старые сначала. |

### Управление участками

| Команда | Описание |
| :--- | :--- |
| `/town claim` | Приватизировать участок, на котором стоите, для города. |
| `/town claim outpost <#\|{имя}\|{имя:#}>` | Приватизировать форпост. `{имя}` — по названию, `{имя:#}` — если имя начинается с цифры. |
| `/town claim {радиус}` | Приватизировать область участков вокруг (радиус от текущей позиции). |
| `/town claim auto` | Автоматически приватизировать максимум участков вокруг, исходя из денег в банке и доступных участков. |
| `/town claim fill` | Заполнить окружённый участками регион внутри границ (после того, как граница захвачена). |

# 🏡 Как работает Towny

## 🧱 Иерархия Towny:

### 🌍 Кочевники (Nomads)

| Характеристика | Описание |
| :--- | :--- |
| **Кто это** | Игроки, не состоящие ни в одном городе. |
| **Земля** | Не имеют собственных участков. |
| **Права** | Настраиваются через `TownyPerms.yml`. |
| **Покупка участков** | Могут купить участок посольства, если имеют право `towny.command.plot.claim`. |
| **Автоматические права** | Все кочевники получают ноду `towny.townless`. |

### 🏠 Жители (Residents)

| Характеристика | Описание |
| :--- | :--- |
| **Кто это** | Игрок, вступивший в город (или автоматически зачисленный при первом входе). |
| **По умолчанию** | При первом заходе получают ноду `towny.town.resident` (прописана в секции nomad файла `townyperms.yml`). |
| **Команда** | `/resident` — открывает экран резидента: баланс, город, участки, друзья. |
| **Авто-город** | Можно задать `default_town_name: 'Название'` в `config.yml`, чтобы новые игроки сразу попадали в конкретный город. |
| **Участки** | Могут приватить участки, выставленные мэром на продажу. При наличии ≥1 участка в `/res` появляется строка с количеством участков и стандартными правами на них (если не заданы индивидуальные). |
| **Права** | Настраиваются в секции `towns.default` файла `TownyPerms.yml`. |

### 🏙️ Города (Towns)

| Характеристика | Описание |
| :--- | :--- |
| **Что это** | Объединение жителей (минимум один) с одним мэром. |
| **Банк** | Город имеет банк, из которого мэр может снимать деньги. |
| **Помощники** | Мэр может назначать ассистентов с аналогичными или ограниченными правами. |
| **Налоги** | Взимаются с жителей в конце каждого «дня» (интервала). |
| **Рост** | Города расширяются от домашнего блока (`homeblock`), захватывая соседние участки, либо создавая аванпосты (`/t claim outpost`). |
| **Ограничения** | Максимальное число жителей можно установить в `global_town_settings.max_residents_per_town` (по умолчанию без ограничений). |
| **Список** | `/town list` показывает все города сервера. |
| **Метаданные** | С версии 0.95.1.0 можно хранить метаданные города. |
| **Продажа города** | С версии 0.99.6.0 город можно выставить на продажу (`/t forsale [$]`), купить (`/t buytown [город]`) или снять с продажи (`/t notforsale`). |

### 👑 Мэры (Mayors)

| Характеристика | Описание |
| :--- | :--- |
| **Кто это** | Лидер города, управляет им вместе с помощниками. |
| **Права** | Настраиваются через `TownyPerms.yml`. |
| **Ранги** | Мэр может назначать жителям ранги (помощник, казначей, строитель и т.д.), созданные админом в `townyperms.yml`. |
| **Назначение рангов** | `/town rank {add\|remove} {игрок} {ранг}`. Один игрок может иметь несколько рангов. |
| **Налоги** | Мэр определяет тип и ставку налогов для города. |

#### 👥 Несколько городов под управлением админа (NPC-мэры)

Администратор может управлять несколькими городами одновременно, используя NPC-мэра.

**Пример: Админ Боб и NPC-город**

- Боб создаёт «Серверный Город», настраивает налоги, цены на участки, права.
- Выдаёт городу неограниченные участки: `/ta town {город} toggle unlimitedclaims`
- Назначает NPC-мэром: `/townyadmin set mayor {город} npc`
- Покидает город: `/t leave`
- Создаёт свой личный город: `/t new {название}`

**Дальнейшее управление:**

- Боб может добавлять игроков в Серверный Город, не покидая свой: `/ta town {город} add {игрок}`, или прописать `default_town_name` в конфиге.
- Настройки города меняются через `/ta town {город} ...`
- NPC-город можно добавить в нацию: `/ta nation {нация} add {город}`
- Создать сразу NPC-город, не выходя из своего: `/ta town new {город} npc`

#### 🏷️ Титулы и суффиксы (с версии 0.95.1.0)

| Команда | Описание |
| :--- | :--- |
| `/town set title {имя} текст` | Добавить титул (префикс) жителю. |
| `/town set surname {имя} текст` | Добавить суффикс (постфикс) жителю. |

### 🚫 Преступники (Outlaws)

| Характеристика | Описание |
| :--- | :--- |
| **Что это** | Список игроков, объявленных вне закона в конкретном городе. |
| **Назначение** | `/town outlaw [add\|remove] [имя]` (требует право `towny.command.town.outlaw`). |
| **Кто может быть** | Любой игрок, не обязательно член города или нации. |
| **Последствия** | — Если преступник был жителем города, его немедленно выгоняют.<br>— В открытый город (`Open`) преступники не могут вступить через `/t join`.<br>— Не могут телепортироваться на спавн публичного города.<br>— При входе в город видят предупреждение в виде заголовка.<br>— Если игрок в сети и его объявили вне закона, он увидит сообщение в чате. |
| **Тюрьма за смерть** | Если преступник умирает в городе, где он вне закона, его могут посадить в тюрьму, если:<br>- `jail.is_jailing_attacking_outlaws: true` в конфиге<br>- убивший имеет право `towny.outlaw.jailer`<br>- в городе есть тюремный участок<br>По умолчанию право `towny.outlaw.jailer` есть у мэра, ассистента и шерифа. |
| **Просмотр списка** | `/town outlawlist` — для своего города.<br>`/town outlawlist {город}` — для любого города.<br>`/resident outlawlist [игрок]` — список городов, где игрок (или вы) вне закона. |
| **Телепортация** (с 0.96.4.0) | Если `allow_outlaws_to_enter_town: false`, преступники телепортируются прочь. Задержка задаётся в `outlaw_teleport_warmup` (0 — мгновенно). |
| **Принудительное PvP** (с 0.101.2.0) | — `force_pvp_on_outlaws: true` — преступников всегда можно атаковать.<br>- `allows_outlaws_to_always_pvp: true` — преступники сами могут атаковать в городах, где они вне закона. |

## 🚩 Нации (Nations)

### Общая информация

| Характеристика | Описание |
| :--- | :--- |
| **Что это** | Объединение городов (минимум один) с одним городом-столицей. |
| **Лидер** | Мэр столицы автоматически становится лидером нации. |
| **Банк** | Нация имеет собственный банк. |
| **Налоги** | Может облагать налогом входящие в неё города. |
| **Война** | Может участвовать в военных событиях и заключать альянсы. |

### Спавн и публичность

| Особенность | Описание |
| :--- | :--- |
| **Спавн нации** | `/nation spawn` — телепортирует на точку спавна нации. |
| **Публичный спавн** | Если нация публичная, на её спавн могут телепортироваться почти все игроки, кроме врагов. |
| **Ограничение спавна** | Можно привязать спавн строго к столице (`global_nation_settings.capital_spawn`), иначе точка может быть где угодно. |

### Альянсы

| Характеристика | Описание |
| :--- | :--- |
| **Что дают** | Защита от friendly-fire, возможность помогать на участках (если разрешено союзникам), совместные военные действия. |
| **Двусторонние альянсы** | С версии 0.91.0.0 можно запретить односторонние альянсы: `war.disallow_one_way_alliance: true` (по умолчанию `false`). |

### Ограничения и требования

| Версия | Настройка | Описание |
| :--- | :--- | :--- |
| 0.91.0.0 | `global_town_settings.minimum_residents_for_nation` | Минимальное число жителей для создания/поддержания нации. |
| 0.96.2.0 | `global_town_settings.maximum_number_residents_without_nation` | Максимум жителей в городе без нации. При >0 городам придётся создавать нации для роста. |

### NationZone (Национальная зона)

| Характеристика | Описание |
| :--- | :--- |
| **Что это** | Зона вокруг городов нации, где строить/ломать могут только члены нации (как обычная дикая местность, но с защитой). |
| **Включение** | `global_nation_settings.nationzone.enable: true` |
| **Размер** | Увеличивается с ростом населения через `NationLevels` в конфиге. Можно сделать зону больше у столицы. |
| **Отключение во время войны** | `global_nation_settings.nationzone.war_disables` |
| **Ручное управление** | Мэр может выключить зону города: `/t toggle nationzone`. Админ может задать размер: `/ta town set nationzoneoverride #`. |

### Расстояния и санкции (с 0.100.1.0)

| Особенность | Описание |
| :--- | :--- |
| **Максимальное расстояние** | Можно ограничить расстояние между столицей и городами, а также между городами внутри нации (см. `proximity` в `Global Nation Settings`). |
| **Санкции городов** | `/nation sanctiontown` — запретить городу вступать в нацию через `/n join` и использовать `/n spawn [нация]` (если нация публичная). |

### Привилегии от роста населения

- Бонусные участки для городов.
- Снижение стоимости содержания города.
- Увеличенная NationZone.
- Повышенный лимит аванпостов.

### Пояснение переменных `town_level`

| Переменная | Описание |
| :--- | :--- |
| `numResidents: 1` | Количество жителей, необходимое городу для достижения данного уровня `town_level`. Может определяться количеством TownBlocks, если `global_town_settings.town_level_is_determined_by_townblock_count_instead_of_resident_count: true`. |
| `namePrefix: ''` | Добавляется **перед** названием города. |
| `namePostfix: ' (Settlement)'` | Добавляется **после** названия города. |

| `mayorPostfix: ''` | Добавляется **после** имени мэра. |
| `townBlockBuyBonusLimit: 0` | Максимальное количество бонусных участков, которые город может купить через `/town buy bonus`. Требует `town.max_purchased_blocks_uses_town_levels: true`. |
| `townBlockLimit: 16` | Переопределяет `town_block_ratio` и задаёт точное количество городских участков. Требует `town_block_ratio: '0'`. |
| `upkeepModifier: 1.0` | Множитель стоимости содержания города. При `town_plotbased_upkeep: true` влияет только если `town_plotbased_upkeep_affected_by_town_level_modifier: true`. |
| `townOutpostLimit: 1` | Лимит аванпостов. Требует `limit_outposts_using_town_and_nation_levels: true`. |
| `debtCapModifier: 1.0` | Множитель лимита задолженности. При `debt_cap_uses_town_levels: true` финальный лимит = `debt_cap.override` × `debtCapModifier`. |
| `peacefulCostMultiplier: 1.0` | Влияет на стоимость нейтралитета города (цена населения × `economy.price_town_neutrality`). |
| `resourceProductionModifier: 1.0` | Изменяет количество ресурсов, ежедневно получаемых городом (плагин TownyResources). |
| `bankCapModifier: 1.0` | Множитель лимита банка города (население × `economy.banks.town_bank_cap`). |
| `townBlockTypeLimits` | Список ограничений на количество участков каждого типа (например, `shop: 2`). Может быть пустым `[]`. |

👉 В примере: при 1 жителе город получает суффикс `(Settlement)`, мэр — префикс `Hermit`. При 2 жителях — `(Hamlet)` и префикс `Chief`.

## 🏗️ Как растут города (How Towns Grow)

### 🚀 Создание города (Starting a Town)

| Характеристика | Описание |
| :--- | :--- |
| **Команда** | `/town new {название}` — создаёт новый город. |
| **Стоимость** | Обычно требует сумму, заданную в `price_new_town`. |
| **Домашний блок** | Городской блок, на котором стоял мэр при создании, становится `homeblock`. Точная позиция — спавн города. |
| **Смена спавна** | `/t set spawn` — переносит точку спавна в пределах домашнего блока. |
| **Смена домашнего блока** | `/t set homeblock` — переносит домашний блок на другой захваченный участок. |
| **Захват участков** | `/town claim` — новые участки должны примыкать к уже захваченным, кроме аванпостов. |
| **Бонус новым городам** | С версии 0.100.2.0 можно выдать разовый бонус участков: `claiming.new_town_bonus_claims` в конфиге. |

### 📥 Вступление в город (Joining Towns)

| Способ | Описание |
| :--- | :--- |
| **По приглашению** | Мэр или ассистент вводит `/town add {игрок}`. Игрок получает запрос и может `/accept` или `/deny`. |
| **Открытый город** | Мэр включает `/town toggle open`. Игрок без города может войти командой `/town join {город}`. Список открытых городов: `/town list by open`. |
| **Эффект** | Новые жители увеличивают лимит участков, доступных мэру для захвата. |

## 📐 Система владения участками (Plot System of Land Ownership)

### 🧱 Городские блоки (Town Blocks)

| Характеристика | Описание |
| :--- | :--- |
| **Концепция** | Защита блоков разбита на участки — townblocks. По умолчанию 16×16×N (N — высота мира). Это равномерная сетка без промежутков. |
| **Захват** | Мэры забирают городские блоки, затем продают/передают их жителям. |
| **Метаданные** | С версии 0.95.1.0 можно хранить метаданные на участке. |

### 📏 Размер городского блока (Town Block Size)

| Настройка | Описание |
| :--- | :--- |
| `town_block_size: 16` | Меняется в `config.yml`. Рекомендуется задавать только при первой установке Towny. Изменение позже сдвинет всё нежелательно. |
| **Меньший размер** | Даёт больше точности, но больше работы. Не рекомендуется меньше 4 (иначе страдает кэширование). |

### 🏁 Захват городских блоков (Claiming Townblocks)

#### Захват для города (Town Claiming)

| Особенность | Описание |
| :--- | :--- |
| **Кто может** | По умолчанию — мэр и ассистенты, имеющие соответствующие права. |
| **Массовый захват** | Требует право `towny.command.town.claim.town.multiple`. |

##### Команды захвата

| Команда | Результат |
| :--- | :--- |
| `/town claim` | Захватывает один участок под игроком. |

| Механика | Описание |

## 🔄 Захват чужих блоков (Overclaiming Townblocks)

| Характеристика | Описание |
| :--- | :--- |
| **Включение** | `town.overclaiming.being_overclaimed_allows_other_towns_to_steal_land: true` (по умолчанию отключено). |
| **Условие для города-жертвы** | Город считается «overclaimed», когда его текущее число блоков превышает лимит (например, 120/100). |
| **Команда захвата** | `/t takeoverclaim` — забрать один соседний блок у перегруженного города. |
| **Требования** | — Использовать на границе своего и чужого города.<br>— Ваш город не должен быть перегружен.<br>— Хватает денег для оплаты (`economy.takeoverclaim.price`).<br>— Возможно, требоваться минимальный возраст города.<br>— Может быть кулдаун. |
| **Уведомления** | Мэр перегруженного города видит предупреждения при входе, при подходе к лимиту, при потере жителей. На доступных для захвата блоках отображается `[TakeoverClaim Available]`. |
| **Ограничения** | Нельзя разделить город на две части — захват идёт снаружи внутрь. Можно защитить радиус вокруг домашнего блока (`town.overclaiming.overclaiming_prevented_by_homeblock_radius`). |
| **Нации и враги** (с 0.100.1.0) | Опционально: захват блоков нации разрешён только врагам (`claiming.overclaiming.nations_required_to_be_enemies`). |

> **Рекомендация:** Лучше всего использовать на серверах с отключёнными аванпостами и требованием более 1 соседнего блока для захвата.

## 🌿 Дикие участки (Wilds Plots)

| Характеристика | Описание |
| :--- | :--- |
| **Назначение** | `/plot set wilds` — создаёт «дикий» участок внутри города. |
| **Что можно делать** | Жителям разрешено добывать блоки из wilds ignore list (руды, деревья, цветы, грибы и т.д.). Нельзя ломать камень, землю, траву и другие ландшафтные блоки. |
| **Применение** | Создание лесных ферм, пещер, защита ландшафта с возможностью собирать ресурсы. |
| **Просмотр разрешённых блоков** | `/towny wildsblocks` — показывает, какие блоки разрешено ломать/использовать на wilds-участках. |

### Настройка Wilds Plots (краткая инструкция)

1. Откройте `towny\data\worlds\WORLDNAME.txt`.
2. Установите:
   - `unclaimedZoneBuild=false`
   - `unclaimedZoneDestroy=false`
3. Настройте `unclaimedZoneIgnoreIds`, добавив нужные блоки.
4. Разметьте wilds-участки командой `/plot set wilds`.
5. По умолчанию жители имеют права build/destroy; можно дополнительно выдать права союзникам или чужакам через `/plot set perm`.

## 🔒 Тюрьма (Jail Plots)

| Характеристика | Описание |
| :--- | :--- |
| **Назначение** | `/plot set jail` — создаёт тюремный участок. |

### Как игроки попадают в тюрьму

- **По команде мэра/шерифа**:
  - **Без залога (Bail Disabled)**:
    - `/town jail [имя]` — сажает на 1 час (только для жителей города).
    - `/town jail [имя] {часы}` — сажает на указанное количество часов.
    - `/town jail [имя] {часы} {тюрьма}` — в конкретный тюремный участок (номер).
    - `/town jail [имя] {часы} {тюрьма} {камера}` — в конкретную камеру (оба параметра — числа).
  - **С залогом (Bail Enabled)**:
    - `/town jail [имя]` — 1 час со стандартной суммой залога.
    - `/town jail [имя] {часы}` — указанные часы, стандартный залог.
    - `/town jail [имя] {часы} {залог}` — часы и сумма залога.
    - `/town jail [имя] {часы} {залог} {тюрьма}` — с указанием тюремного участка.
    - `/town jail [имя] {часы} {залог} {тюрьма} {камера}` — с указанием и участка, и камеры.
- **За смерть врага в городе**: атакующий, считающийся врагом (Enemy нации), после смерти попадает в `primaryjail` защищающегося города.
- **За смерть преступника (Outlaw)**: если `jail.is_jailing_attacking_outlaws: true`, и убивший имеет право `towny.outlaw.jailer`, преступник попадает в primaryjail.

### Освобождение из тюрьмы

- Игрок покидает город и становится кочевником.
- Мэр/шериф использует `/town unjail [имя]`.
- Заключённый платит залог: `/resident jail paybail`.
- Игрок сбегает из тюремного участка и города в дикую местность.

### Дополнительные механики тюрем

| Механика | Описание |
| :--- | :--- |
| **Запрет телепортации** | Заключённые не могут телепортироваться. |
| **Жемчуг Края/Хорус** | Запрещены, если не разрешено в конфиге. |
| **Смерть в тюрьме** | Возвращает обратно в свою тюремную камеру. |
| **Деньги за убийство** | С заключённых не выплачиваются. |
| **Статус** | Отображается в `/res [имя]` с указанием города-тюремщика. |
| **Ранг Шерифа** | Рекомендуется создать ранг `Sheriff` в `townyperms.yml` с правом `towny.command.town.toggle.jail`. |
| **Чёрный список команд** | `jail.blacklisted_commands` — запрещённые для заключённых команды. |
| **Выход из игры** | Опционально: заключённый, вышедший до телепортации в камеру, погибает. Игрок предупреждается. |
| **Иммунитет новичков** | Можно задать время в конфиге, в течение которого новые игроки не могут быть посажены. |

### Ячейки и управление тюрьмой

| Команда / Действие | Описание |
| :--- | :--- |
| `/plot jailcell add` | Добавить ячейку в тюремном участке (спавн помечается кольцом частиц). |
| `/plot jailcell remove` | Удалить ячейку. |
| `/town jail list` | Список всех тюрем города. |
| `/town set primaryjail` | Установить основную тюрьму (стоя в jail-участке). Первая созданная тюрьма становится основной автоматически. |
| **Книга при посадке** | Заключённый получает книгу с правилами и способами выхода (информация из конфига). |

### Система залога (Bail System)

| Настройка | Описание |
| :--- | :--- |
| `jail.bail.is_allow_bail: true` | Включает возможность залога. |
| **Сумма залога** | Может указываться при команде `/town jail`. Максимальная сумма задаётся в `jail.bail.bailmax_amount`. |
| **По умолчанию** | Для Outlaws, врагов, мэров и королей действуют стандартные суммы залога. |

### Отключение тюрем в мире

| Команда | Описание |
| :--- | :--- |
| `/tw toggle jailing` | Включает/отключает возможность сажать в тюрьму в конкретном мире. |

## 🌾 Ферма (Farm Plots)

| Характеристика | Описание |
| :--- | :--- |
| **Назначение** | `/plot set farm` — создаёт фермерский участок. |
| **Разрешённые действия** | Только ломать/ставить блоки из списка `global_town_settings.farm_plot_allow_blocks`. |
| **Просмотр блоков** | `/towny allowedblocks` — показывает список разрешённых фермерских блоков. |
| **Права** | Определяются строкой прав (`/plot perm`). Например, при `B=rnao` строить могут все, при `B=r---` только жители города. |
| **Пашня (Farmland)** | Можно добавить `FARMLAND` в список разрешённых блоков, чтобы игроки могли создавать грядки мотыгой. По умолчанию не включено, создавать пашню могут только админы/мэры/ассистенты. Вытаптывание farmland защищено Towny. |
| **Убийство животных** | Разрешено убивать животных из списка `global_town_settings.farm_animals` (по умолчанию: PIG, COW, CHICKEN, SHEEP, MOOSHROOM), если игрок может ломать пшеницу на участке. |

## 🏦 Банк (Bank Plots)

| Характеристика | Описание |
| :--- | :--- |
| **Назначение** | `/plot set bank` — создаёт банковский участок. |
| **Ограничения** | Можно ограничить взносы/снятия из банка города и нации только банковскими участками. Включается в `bank.is_banking_limited_to_bank_plots`. |
| **Принудительное использование** | Если в городе есть хотя бы один bank-участок, можно запретить банковские операции через homeblock: `bank.do_homeblocks_not_work_when_a_town_has_bank_plots: true`. |
| **Другие плагины** | Могут использовать банковские участки Towny для своих механик. |
